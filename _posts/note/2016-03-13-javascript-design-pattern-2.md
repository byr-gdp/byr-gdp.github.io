# 第12章 享元模式

1. 享元的核心是运用共享技术来有效支持大量细粒度的对象
2. 如何划分内部状态和外部状态（P167）
3. 享元模式是一种用时间换空间的优化模式
4. 当对象没有内部状态的时候，生产共享对象的工厂实际上变成了一个单例工厂。
5. 对象池是另外一种性能优化方案，跟享元模式有一些相似之处，但没有分离内部状态和外部状态的过程。
